# mf-dev

*mf-dev* is a set of tools to facilitate easy development of micro-frontends using Vite and some of the most common frameworks

## Basic Usage
### Custom element wrapper

```
React:
@mf-dev/wrapper-react

Vue:

Svelte:

SolidJS:
@mf-dev/wrapper-solid
```
